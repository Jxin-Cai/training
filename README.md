# Training

这是一个 AI 辅助开发方法论的多语言示例项目仓库。

## 简介

本仓库包含四个主要的 AI 辅助开发方法论示例项目，每个项目都提供 Java、Python、C++ 三种语言的实现案例。

## 项目结构

```
training/
├── 1.riper-5/              # RIPER-5 方法论示例
│   ├── java/               # Java 实现
│   ├── python/             # Python 实现
│   └── cpp/                # C++ 实现
│
├── 2.bmad/                 # BMAD 方法论示例
│   ├── java/               # Java 实现
│   ├── python/             # Python 实现
│   └── cpp/                # C++ 实现
│
├── 3.spec-kit-openspec/    # Spec-Kit + OpenSpec 示例
│   ├── java/               # Java 实现
│   ├── python/             # Python 实现
│   └── cpp/                # C++ 实现
│
└── 4.omo-superpowers/      # OMO + Superpowers 示例
    ├── java/               # Java 实现
    ├── python/             # Python 实现
    └── cpp/                # C++ 实现
```

## 方法论说明

### 1. RIPER-5
RIPER-5 是一个提示词工程方法论，用于提升 AI 辅助编程的效果。

### 2. BMAD
BMAD (Behavior-driven Multi-Agent Development) 是一个行为驱动的多智能体开发方法论。

### 3. Spec-Kit + OpenSpec
Spec-Kit 是一个规范工具集，OpenSpec 是一个开放规范标准，两者结合可以实现高质量的软件开发。

### 4. OMO + Superpowers
OMO (One More Optimization) 结合 Superpowers 技能系统，提供强大的 AI 辅助开发能力。

## 快速开始

1. 选择你感兴趣的方法论项目
2. 进入对应的语言目录
3. 查看该目录下的 README.md 了解具体示例

## 环境要求

### Java 项目
- JDK 11 或更高版本
- Maven 或 Gradle

### Python 项目
- Python 3.8 或更高版本
- pip

### C++ 项目
- C++17 或更高版本
- CMake 3.10 或更高版本
- GCC/Clang/MSVC
